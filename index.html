Sobre o Projeto

Este projeto foi desenvolvido como parte de um Desafio de Projeto para praticar HTML. A seguir, você verá uma demonstração de várias tags HTML que aprendi.

Vamos explorar tags básicas e novas:

  • Tags básicas como <h1> até <h6>
  • Tags de formatação como <p>, <strong>, <i>
  • Novas tags como <font>, <del>, <abbr>
